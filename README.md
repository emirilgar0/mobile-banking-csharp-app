Mobile Banking App - Currency Exchange Module
Bu proje, C# Windows Forms kullanılarak geliştirilmiş, SQL Server veritabanı entegrasyonlu bir bankacılık döviz işlem modülüdür. Kullanıcıların TL ve USD hesapları arasında güncel kurlar üzerinden güvenli bir şekilde alış/satış yapmasına olanak tanır.

Öne Çıkan Teknik Özellikler
ADO.NET & SQL Server: Hesap bakiyeleri veritabanı üzerinden anlık olarak kontrol edilir ve güncellenir.

İşlem Güvenliği: Döviz alım/satım öncesinde bakiye kontrolü yapılır; yetersiz bakiye durumunda işlem engellenir.

Otomatik Loglama: Her başarılı işlem sonrası HesapÖzeti tablosuna otomatik olarak işlem detayları (tarih, tutar, açıklama) kaydedilir.

Custom UI Logic: ComboItem sınıfı ile ComboBox nesneleri içinde hem metin (hesap no) hem de ID değerleri profesyonelce yönetilmiştir.


Teknolojiler
Dil: C#

Framework: .NET Framework / WinForms

Veritabanı: SQL Server (ADO.NET)
